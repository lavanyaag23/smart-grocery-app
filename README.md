# Smart Grocery App

A frontend grocery shopping application built using React.

## Features

* User Login
* Grocery Product Listing
* Shopping Cart
* Checkout System
* Admin Dashboard
* Responsive UI

## Technologies Used

* React.js
* HTML
* CSS
* JavaScript

## Installation

Clone the repository:

```bash
git clone https://github.com/lavanyaag23/smart-grocery-app.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```
